📁 File Setup

Keep this index.html in the same folder as all the other tool files. The links above use relative paths, so they'll break if tools are moved to a different folder.

📄 CSV Source

All tools take a CSV exported from the BLAST platform. Use either a single-tournament file or your full master database — tools will let you pick the event after import.

🖨️ Printing

When printing PDFs from any tool, enable Background Graphics in your browser's print dialog. Without it, colored backgrounds and charts won't show up.
